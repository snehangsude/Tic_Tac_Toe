# Tic_Tac_Toe
A console game of Tic-Tac-Toe

### Written in python 3.9 

## Installation on Windows:
* `git clone https://github.com/snehangsude/Tic_Tac_Toe.git`
* `cd Tic_Tac_Toe`
* `python main.py`

## Installation on Linux and Unix like OS:
* `git clone https://github.com/snehangsude/Tic_Tac_Toe.git`
* `cd Tic_Tac_Toe`
* `python3 main.py`
